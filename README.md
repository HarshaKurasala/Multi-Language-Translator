# 🌍 WorldSpeakAI - Intelligent Multilingual Translator

**WorldSpeakAI** is a modern, responsive, and highly intuitive web application designed to break down language barriers through advanced AI-powered translation. Built with a focus on user experience, it combines a sleek "human-made" aesthetic with powerful features like real-time voice-to-text and native speech synthesis.

## ✨ Features

-   **🚀 Robust AI Translation:** Leverages the **MyMemory API** with a seamless **Google Translate fallback** to ensure your translations are always accurate and available.
-   **🎙️ Voice Input (Speech-to-Text):** Hands-free translation! Click the microphone to speak, and the app will capture your voice and translate it instantly.
-   **🔊 Text-to-Speech (Listen):** Hear the correct pronunciation for both your input and the translated output in native accents.
-   **🇮🇳 Expanded Indian Language Support:** Includes full support for **Telugu, Marathi, Tamil, Gujarati, Hindi, and Bengali**.
-   **🎨 Modern "Human-Made" UI:** Features a sophisticated Indigo & Slate color palette, smooth entry animations, glassmorphism elements, and animated background blobs for a premium feel.
-   **🔄 Smart Language Swap:** Instantly flip your source and target languages (including text content) with a single click.
-   **📋 Clipboard Integration:** Copy your results with one click using our integrated feedback system.
-   **📱 Fully Responsive:** Optimized for everything from large desktop monitors to compact smartphones.

## 🛠️ Tech Stack

This project was built using pure web technologies to ensure maximum performance and zero dependencies:

-   **Frontend:** HTML5, CSS3 (Grid & Flexbox, Custom Animations)
-   **Logic:** Vanilla JavaScript (ES6+)
-   **Icons:** Font Awesome 6.4.0
-   **Typography:** Inter (via Google Fonts)
-   **APIs:** 
    -   MyMemory Translation API
    -   Google Translate (Gtx)
    -   Web Speech API (SpeechRecognition & Synthesis)

## 🚀 Getting Started

No installation, no build steps, and no API keys required!

1.  **Clone or Download** this repository.
2.  Navigate to the project folder.
3.  Open `index.html` in your favorite web browser (we recommend **Microsoft Edge** or **Google Chrome** for the best voice support).
4.  Start translating!

## 💡 How it Works

1.  **Select Languages:** Choose your source (or use Auto-Detect) and your target language.
2.  **Input Text:** Type manually or click the **Microphone (🎙️)** to speak.
3.  **Translate:** Hit the **Translate** button. The app will show a loading spinner and then provide the result in emerald green.
4.  **Interact:** Listen to the pronunciation, copy to your clipboard, or swap languages for a reply.
